=======================
  BATTLESHIP LITE APP
=======================

Basic Requirements
------------------

A two-player console game that has its roots in the game Battleship from Mattel. There will be a 25-spot grid (A1 - E5). Each Player will place five pegs on the board to represent their five ships. Player will then take turns firing on their opponent's ships. The first person to sink all five ships wins.


Other details
-------------

- Each ship take up one spot on the grid
- Two player will share the same console
- No completing the round after 5 sunk ships
- Show the visual of the grid with hits and misses
- Don't allow user to shoot the same spot twice


Grid Display
------------

A1 A2 A3 A4 A5
B1 B2 B3 B4  X
C1 C2 C3 C4 C5
 O  O D3 D4 D5
 X  E2 X E4 E5

<Letter><Number> - empty spot
X - Hit
O - Miss
